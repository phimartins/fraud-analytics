Data Quality Report e Deteco de Fraudes em Transaes Imobilirias

</br>

Etapa 1 - Data Quality Report (DQR)

</br>

O DQR um relatrio analtico com o objetivo de compreender a organizao dos dados, se esto coerentes, se h alguma anomalia amplamente visvel e resumir os dados (ou pelo menos as variveis mais importantes) com base na compreenso do problema de negcio.

</br>

Para o projeto, foi utilizado uma base de dados fornecida pelo portal de dados abertos da cidade de Nova Yorque.


Voc pode acessar o Dicionrio de Dados e fazer download do dataset clicando aqui.

# Sumrio estatstico das variveis numricas
summ_num = pd.DataFrame(index = df_num.columns)
summ_num['Tipo de Dado'] = df_num.dtypes.values
summ_num['# Registros No Nulos'] = df_num.count().values
summ_num['# Registros No Zero'] = df_num.astype(bool).sum(axis = 0)
summ_num['% Populado'] = round(summ_num['# Registros No Nulos'] / df_num.shape[0]*100,2)
summ_num['# Valores nicos'] = df_num.nunique().values
summ_num['Mean'] = round(df_num.mean(),2)
summ_num['Std'] = round(df_num.std(),2)
summ_num['Min'] = round(df_num.min(),2)
summ_num['Max'] = round(df_num.max(),2)
